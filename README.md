# MURRINHA — O Gazeador 🦁🐦

Um jogo de plataforma 16-bits (estilo Super Mario World) sobre **gazear aula**
no centro de Campina Grande — PB. Praça da Bandeira, CAD, pombos, Liginha,
trombadinhas e o ônibus lotado de volta pra casa.

**Jogue no browser, instale como app (PWA) e jogue em tela cheia no celular.**

## Como jogar

| Ação | Teclado | Celular |
|---|---|---|
| Andar | ← → | D-pad virtual |
| Pular | Z ou Espaço | Botão **A** |
| Correr | X ou Shift | Botão **B** |
| Pausar | Enter | Toque no canto superior direito |

- Colete **fichas telefônicas** (100 = vida extra)
- **Chocolate das Brasileiras** = +1 vida · **Pipoca do Galego** = turbo
- Pise nos trombadinhas; da **Liginha**, só correndo
- Não pare embaixo de pombo. Sério.

## Rodar localmente

```
node tools/serve.mjs
# abre http://localhost:8123
```

Sem build, sem dependências: HTML + Canvas + JavaScript puro.
Pixel art e música chiptune 100% geradas em código.
